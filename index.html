What this system does
ZagaPrime Sales System is a lightweight internal CRM for finding local businesses with weak or missing websites, importing them into your pipeline, tracking every touchpoint, saving repeat searches, and moving each lead from first review to closed-won. Your working data stays in local browser storage for speed, while Google Sheets gives you a backup and sync layer.

Live Search Deployment
Live lead search runs through the Vercel serverless route at /api/search-leads. You do not need a Claude API key. Add GOOGLE_PLACES_API_KEY in your Vercel project environment variables, then redeploy so the no-website search can query Google Places.

One-Time Setup: Google Apps Script Web App
This enables full outbound sync with the payload shape expected by the web app: type, leads, activity, and synced_at.
1
Open your Google Sheet, then go to Extensions > Apps Script.
2
Delete any existing code and paste the script shown below.
3
Deploy as a web app with Execute as: Me and Who has access: Anyone.
4
Paste the deployed /exec URL below and click Test Connection.
Apps Script Code
Copy this directly into Google Apps Script if you want the dashboard to push full lead and activity data into Sheets.
const SHEET_ID = '1Sb4_mejZZR7UuOwzc3a6CK5gjfF5NeX__g6zN0G_boM';
const LEADS_SHEET_NAME = 'Sheet1';
const ACTIVITY_SHEET_NAME = 'Activity';

function doPost(e) {
  try {
    const data = JSON.parse(e.postData.contents);
    const ss = SpreadsheetApp.openById(SHEET_ID);

    if (data.type === 'sync_leads') syncLeads(ss, data.leads);
    if (data.type === 'sync_activity') syncActivity(ss, data.activity, data.leads);
    if (data.type === 'full_sync') {
      syncLeads(ss, data.leads);
      syncActivity(ss, data.activity, data.leads);
      formatSheets(ss);
    }

    return ContentService.createTextOutput(
      JSON.stringify({ success: true, timestamp: new Date().toISOString() })
    ).setMimeType(ContentService.MimeType.JSON);
  } catch (err) {
    return ContentService.createTextOutput(
      JSON.stringify({ success: false, error: err.message })
    ).setMimeType(ContentService.MimeType.JSON);
  }
}

function doGet() {
  return ContentService.createTextOutput(
    JSON.stringify({ status: 'ZagaPrime LeadFunnel webhook is live', time: new Date().toISOString(), sheet: SHEET_ID })
  ).setMimeType(ContentService.MimeType.JSON);
}
